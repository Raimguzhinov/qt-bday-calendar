Добавление заметки


При выборе друга из левой таблицы появляется возможность изменить описание события этого дня рожения. Для этого надо ввести заметку и нажать на кнопку "Добавить" или клавишу Enter. Информация о данной заметке будет автоматически заполнена. После чего в правом редакторе можно увидеть обновленную информацию заметки по этому собитию дня рожения.
